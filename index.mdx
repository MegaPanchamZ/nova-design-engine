# Nova Design Engine

Nova Design Engine is an installable toolkit for building design editors, AI-assisted layout flows, and export pipelines without locking your app to one model provider.

## What You Get

- Core scene graph types and geometry/layout utilities.
- AI turn orchestration through a provider-agnostic `NovaLLMBinding` interface.
- Prebuilt React editor components from `nova-design-engine/react`.
- Browser-only export helpers from `nova-design-engine/browser`.
- Parser-safe imports from `nova-design-engine/parser`.
- Official adapters for OpenAI and Google GenAI.

## Install

```bash
npm install nova-design-engine
```

If you plan to render the editor UI, install `react` and `react-dom` in the host app as well.

## Minimal Example

```ts
import { runNovaTurn, type NovaLLMBinding } from 'nova-design-engine';

const binding: NovaLLMBinding = {
  async complete(input) {
    return '[MESSAGE]Ready[/MESSAGE][HTML]<div id="root"></div>[/HTML][TWEAKS][][/TWEAKS]';
  },
  async generateImage(prompt) {
    return '';
  },
};

const result = await runNovaTurn(binding, {
  prompt: 'Create a modern pricing card',
  history: [],
  contextNodes: [],
  selectedIds: [],
  basePosition: { x: 80, y: 80 },
});
```

## Security Model

Keep provider API keys out of browser code. The React package does not read `NEXT_PUBLIC_*` secrets, and `NovaAI` stays disabled until your app explicitly supplies a secure binding.
